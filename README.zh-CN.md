> 这是一个基于 `Vue3` 的模版,希望能给予你一定的帮助.

### 介绍

- 集成了 `Axios` , 并进行了简单的封装, 便于使用
- 封装了一个简易的前端错误请求异常捕获功能, 有助于你快速定位问题
- 全面支持 `Typescript`
- 配置了 `Prettier` , 提供更加规范的开发模式, 可通过指令去运行它
- 安装了 `Nprogress` , 更加平滑的加载动画
- 集成了 `Vue-Router` , 合理的结构化路由配置, 让你更加安心的去管理它
- 集成了 `Pinia`

### 安装

```shell
pnpm install
```

```shell
# start
pnpm run dev

# build
pnpm run build

#prettier
pnpm run prettier
```
